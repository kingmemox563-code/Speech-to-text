PROJE SİSTEM VE API BİLGİLERİ

Bu proje aşağıdaki sistemleri ve API servislerini kullanmaktadır:

1. KULLANILAN SİSTEMLER:
   - OpenAI Whisper (Transkripsiyon - STT)
   - OpenAI GPT-4o (Metin Analizi ve Özetleme)
   - OpenAI TTS-1 (Sesli Yanıt - TTS)
   - Google Gemini 1.5 Flash (Metin Analizi - Alternatif)
   - FFmpeg (Ses İşleme Altyapısı)

2. GEREKLİ API ANAHTARLARI:
   Projenin tam kapasite çalışabilmesi için `.env` dosyasına şu anahtarlar eklenmelidir:
   
   OPENAI_API_KEY=SİZİN_OPENAI_ANAHTARINIZ
   GEMINI_API_KEY=SİZİN_GEMINI_ANAHTARINIZ

NOT: Proje ile birlikte bir `.env.example` dosyası sunulmuştur. Bu dosyayı `.env` olarak adlandırıp kendi anahtarlarınızı ekleyebilirsiniz.
